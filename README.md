# Salary Prediction using Linear Regression

## Project Overview
This project demonstrates Linear Regression using the Salary Dataset.

The model predicts employee salary based on years of experience.

This project helps understand basic regression concepts in Machine Learning.

---

# Dataset Used

## Features
- YearsExperience

## Target Variable

Since salary is a continuous numerical value, this is a regression problem.

# Working of Linear Regression
Linear Regression finds the best-fit line between:
- Input variable (Years of Experience)
- Output variable (Salary)

Equation of line:

y = mx + c

Where:
- y = Predicted Salary
- x = Years of Experience
- m = Slope
- c = Intercept

---

# Output
The model predicts salary based on experience.

Example:
- Experience = 5 years
- Predicted Salary = 75000
